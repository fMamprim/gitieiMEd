INSTALLING THE MAP...

This map is very very easy to install...

You don't need to make backup, i already put a backup on the 
map ZIP LOL...
Well...

Just copy the files on the ''Family Park'' folder on this 
directory of your GTA SAN ANDREAS:

C:\Program Files\Rockstar Games\GTA San Andreas\data\maps\LA\

Start a new game or a save game and then enjoy!...


LOCATION:
Skate Park next to the LS (Los Santos) Hospital.
