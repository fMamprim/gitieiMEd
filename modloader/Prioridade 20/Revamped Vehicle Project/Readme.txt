NOTES
Recommended using it together with latest version of VehFuncs, ImVehFt and SkyGfx.
Set [vehiclePipe= "Mobile","Neo" or "Env"] inside skygfx1.ini to preserve the chrome details and avoid transparency bugs.

-Added several fixes and improvement
-Compatible with Proper Vehicle Retex
